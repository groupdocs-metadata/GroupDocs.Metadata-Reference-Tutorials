## 常见问题

**Q: 我能加载受密码保护的 PDF 并仍然读取其 metadata 吗？**  
A: 可以。初始化加载器时提供密码，即可访问 metadata。

**Q: 该库支持读取加密 ZIP 压缩包的 metadata 吗？**  
A: 它可以读取标准 ZIP 的 metadata；对于加密压缩包，需要提供解密密码。

**Q: GroupDocs.Metadata 如何处理大文件？**  
A: 该库采用流式处理，即使是多千兆字节的文档，内存消耗也保持低水平。

**Q: 能否一次性删除文件中的所有 metadata？**  
A: 可以，`clean()` 方法会删除所有受支持的 metadata 属性，同时保留原始内容。

**Q: 哪些 Java 版本兼容？**  
A: 完全支持 Java 8 及更高版本。
