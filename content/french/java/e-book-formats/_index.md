# Mettre à jour les métadonnées EPUB Java – Tutoriels sur les métadonnées des livres numériques

Si vous devez modifier de manière programmatique les informations contenues dans les fichiers EPUB, vous êtes au bon endroit. Dans ce guide, nous vous montrerons **how to update EPUB metadata java** en utilisant la puissante bibliothèque GroupDocs.Metadata.
